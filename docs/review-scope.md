## 先生へ伝える説明文案

> 本リポジトリは、卒業制作 Metis の設計レビュー用に、元リポジトリから必要資料を抽出・再構成したものです。  
> 元リポジトリには全画面・全ER・全ADRを含む詳細設計がありますが、初回レビューでは量が多いため、特に `05-component-tree` 以降の設計図を中心に整理しました。  
> まずは、コア機能である「ゴール入力 → ガイド生成 → ガイド進行 → AI相談・評価 → 教材化 → 教材閲覧」について、画面分割・データ要求・ER・DB方針がつながっているかをご確認いただきたいです。

## Actionsで画像を取得するためのSecret

このレビューリポジトリはPrivateの `5B-Projects/metis` から画像を取得するため、以下のSecretが必要です。

| Secret名 | 用途 |
|---|---|
| `METIS_REPO_TOKEN` | `5B-Projects/metis` を読み取るための GitHub token |

Fine-grained personal access tokenを使う場合は、対象リポジトリを `5B-Projects/metis` に限定し、権限は `Contents: Read-only` で足ります。
